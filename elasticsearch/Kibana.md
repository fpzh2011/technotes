# Kibana概要

Kibana 按时间过滤：
```
@timestamp: [2017-11-23T07:13:06.037Z TO 2017-11-23T09:13:07.037Z]
```
注意：筛选条件中的时间是 UTC 时间，如果日志中是北京时间，会差8个小时。
